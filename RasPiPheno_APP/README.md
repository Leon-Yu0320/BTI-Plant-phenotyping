# **RaspiPheno APP**

RasPiPheno App is developed by [**Julkowska**](https://btiscience.org/magda-julkowska/) and [**Nelson**](https://btiscience.org/andrew-nelson/) lab at the [**Boyce Thompson Institute, Cornell University**](https://btiscience.org/).

The App is a part of high-throughput phenotypic data processing system and it aimed to streamline the downstream phenotypic data collected by customized phenotypic facilities which include [**PhenoRig and PhenoCage**](https://www.protocols.io/edit/bti-mobile-plant-phenotyping-system-phenotyping-fa-cavmse46)

## User manual
To use the APP, please visit our [**host server**](https://github.com/Leon-Yu0320/BTI-Plant-phenotyping/edit/main/RasPiPheno_APP/) or download all files under the current repository to excute the program, more details and instruction of APP usage can be reached at [**Shiny R**](https://shiny.rstudio.com/articles/running.html) website.
